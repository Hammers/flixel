function add(Graphic:Dynamic, Unique:Bool, Key:String):CachedGraphics

Loads a bitmap from a file, caches it, and generates a horizontally flipped version if necessary.

@param Graphic The image file that you want to load. @param Unique Ensures that the bitmap data uses a new slot in the cache. @param Key Force the cache to use a specific Key to index the bitmap. @return The BitmapData we just created.

function checkCache(Key:String):Bool

Check the local bitmap cache to see if a bitmap with this key has been loaded already.

@param Key The string key identifying the bitmap. @return Whether or not this file can be found in the cache.

function create(Width:Int, Height:Int, Color:Int, Unique:Bool, Key:String):CachedGraphics

Generates a new BitmapData object (a colored square) and caches it.

@param Width How wide the square should be. @param Height How high the square should be. @param Color What color the square should be (0xAARRGGBB) @param Unique Ensures that the bitmap data uses a new slot in the cache. @param Key Force the cache to use a specific Key to index the bitmap. @return The BitmapData we just created.

function dumpCache():Void

Dumps bits of all cached graphics. This restores memory, but you can't read / write pixels on those graphics anymore. You can call onContext() method for each CachedGraphic object which will restore it again.
