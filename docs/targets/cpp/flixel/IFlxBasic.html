FlxBasic

This is a useful "generic" Flixel object. Both FlxObject and * FlxGroup extend this class. Has no size, position or graphical data.

This class is for FlxTypedGroup to work with interface instead of FlxBasic, which is needed so that FlxSpriteGroup could extend FlxTypedGroup and be typed with IFlxSprite
